
### 1. Operadores Aritméticos

Usados para cálculos matemáticos.

- **Adição:** `+` (Ex: `5 + 3 = 8`)
    
- **Subtração:** `-` (Ex: `10 - 4 = 6`)
    
- **Multiplicação:** `*` (Ex: `2 * 6 = 12`)
    
- **Divisão:** `/` (Retorna **sempre** um float. Ex: `10 / 3 = 3.333...`)
    
- **Divisão Inteira (Piso):** `//` (Retorna a parte inteira do quociente. Ex: `10 // 3 = 3`)
    
- **Módulo (Resto da Divisão):** `%` (Útil para verificar paridade, por exemplo. Ex: `10 % 3 = 1`)
    
- **Exponenciação:** `**` (Ex: `2 ** 3 = 8`)
    

### 2. Operadores de Atribuição

Usados para atribuir valores a variáveis, muitas vezes combinados com operadores aritméticos.

- **Atribuição Simples:** `=` (Ex: `x = 10`)
    
- **Adição e Atribuição:** `+=` (Equivalente a `x = x + 5`)
    
- **Subtração e Atribuição:** `-=` (Equivalente a `x = x - 2`)
    
- **Multiplicação e Atribuição:** `*=` (Equivalente a `x = x * 4`)
    
- **Divisão e Atribuição:** `/=` (Equivalente a `x = x / 2`)
    

### 3. Operadores de Comparação (Relacionais)

Retornam um valor booleano (`True` ou `False`). Cruciais para estruturas de controle (como `if`).

- **Igual a:** `==` (Ex: `a == b`)
    
- **Diferente de:** `!=` (Ex: `a != b`)
    
- **Maior que:** `>`
    
- **Menor que:** `<`
    
- **Maior ou igual a:** `>=`
    
- **Menor ou igual a:** `<=`
    

### 4. Operadores Lógicos

Usados para combinar expressões booleanas. Importantes para lógica de programação e automação.

- **AND:** `and` (Retorna `True` se **ambas** as expressões forem `True`)
    
- **OR:** `or` (Retorna `True` se **pelo menos uma** das expressões for `True`)
    
- **NOT:** `not` (Inverte o valor booleano. Ex: `not True` é `False`)
    

### 5. Operadores de Identidade

Verificam se dois objetos são o **mesmo objeto na memória**.

- **É:** `is` (Verifica se os objetos são o mesmo)
    
- **Não é:** `is not` (Verifica se os objetos não são o mesmo)
    
    - _Nota:_ Diferente de `==`, que verifica se os **valores** são iguais.
        

### 6. Operadores de Pertença (Membership)

Verificam se um valor está presente em uma sequência (lista, tupla, string, etc.).

- **Em:** `in` (Ex: `'devops' in curso`)
    
- **Não em:** `not in`
    

### 7. Operadores Bitwise (Bit a Bit)

Operam em bits individuais dos operandos (Mais avançados, mas úteis em otimizações ou baixo nível).

- **AND Bit a Bit:** `&`
    
- **OR Bit a Bit:** `|`
    
- **XOR Bit a Bit:** `^`
    
- **Deslocamento à Esquerda:** `<<`
    
- **Deslocamento à Direita:** `>>`
